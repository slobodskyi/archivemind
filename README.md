# ArchiveMind

AI-powered creator archive workspace — an infinite-canvas photo archive for
documentary photographers and photojournalists whose files are scattered across
local disks, Google Drive, and Dropbox. It unifies sources into one workspace,
runs AI analysis (tags, multilingual captions, attribute-level recognition), and
offers composite natural-language search that combines EXIF metadata (where/when)
with AI content understanding (what) — e.g. *"mustached men I shot in Odesa."*

## Status

**Live in production** (Phases 0–1 shipped, Phase 2 in progress — see
[`docs/PLAN.md`](docs/PLAN.md)): [`apps/web`](apps/web) on Vercel (real auth,
drag-and-drop upload to R2, canvas renders your actual archive),
[`apps/worker`](apps/worker) on Railway (ingest: dedup/EXIF/previews incl.
HEIC/RAW; analyze: Gemini tags + embeddings, user-triggered),
[`packages/shared`](packages/shared) (zod contracts). Map/Sense/projects/search
still run on mock data until their phases.

## Getting started

```bash
pnpm install
pnpm dev           # http://localhost:3000 (turbo run dev)
```

Other commands (all must pass before merging):

```bash
pnpm build         # production build (turbo run build)
pnpm lint          # ESLint (turbo run lint)
pnpm typecheck     # tsc --noEmit, strict (turbo run typecheck)
```
